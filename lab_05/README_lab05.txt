lab05 - Biblioteca statica pentru clasele de date (core) si folosirea ei in engine


1) Pentru a compila (folosind CMake):


mkdir build && cd build

cmake ..

cmake --build .


2) Executabilul va aparea in folderul build/


3) Pasii pentru Git:


git checkout -b lab05

git add .
git commit -m "lab05: added static core library and engine"
git push -u origin lab05
